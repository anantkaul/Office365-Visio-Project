# Deploy Visio Professional 2024 LTSC
To Install Visio Professional 2024: (Windows only)

1. [Download](https://github.com/anantkaul/VisioPro2024/archive/refs/heads/main.zip) this repository.
2. Extract & Rename folder `VisioPro2024-main` to `VisioPro2024`
3. Move `VisioPro2024` to `C:`
4. Run CMD as administrator and execute the following commands to install:
```sh
	cd C:\VisioPro2024
	.\setup.exe /configure .\Configuration.xml
```
4. Wait for the setup to install. It may take a few minutes to install.
5. Enjoy your free Visio Professional 2024.
